# CEFR levels and text adaptation

The user picks one of: `a1`, `a2`, `b1`, `b2`, `c1`, `c2` (lowercase
in the file path).

## Adaptation targets

| Level | Word count | Sentences | Grammar                                        | Vocabulary                       |
| ----- | ---------- | --------- | ---------------------------------------------- | -------------------------------- |
| `a1`  | 60–120     | short     | present tense, "to be", "to have"              | ~500 most common words           |
| `a2`  | 120–220    | short     | + past simple, simple future, modal "can"      | ~1 000 most common words         |
| `b1`  | 220–320    | medium    | + conditionals, perfect tenses, passive (light) | ~2 000 most common words        |
| `b2`  | 320–450    | medium    | + reported speech, hypotheticals, full passive | ~4 000 words, some idioms        |
| `c1`  | 450–600    | mixed     | + subjunctive, complex subordination            | ~8 000 words, idioms, register  |
| `c2`  | 600–800    | any       | full native range                              | ~12 000+ words, nuance, idioms   |

## How to adapt

1. Read the source.
2. Identify its level (rough estimate).
3. If source level ≤ target level, lightly edit for clarity and length.
4. If source level > target level:
   - Replace harder words with simpler synonyms.
   - Break long sentences into shorter ones.
   - Drop digressions; keep main facts.
   - Cut to the target word count.
5. Always keep:
   - Topic and core meaning
   - Proper nouns (names, places, organisations)
   - At least one date/number for context
6. Always produce coherent prose, not bullet points.

## Vocabulary count for the lesson

| Level    | Vocabulary items (`vocabulary` block) |
| -------- | --------------------------------------------- |
| `a1`–`a2` | 15–25                                        |
| `b1`–`b2` | 20–35                                        |
| `c1`–`c2` | 25–40                                        |

Choose words that are central to the text and not already known at lower
levels.
